# DevOps Capstone Project

![CI Build](https://github.com/srijoni19/devops-capstone-project/actions/workflows/main.yml/badge.svg)

## Overview

This project implements a Customer Accounts Microservice for an e-commerce platform. The microservice provides RESTful APIs to manage customer account information, including customer names and addresses.

The service supports the following operations:

* Create a customer account
* Read a customer account
* Update a customer account
* Delete a customer account
* List all customer accounts

## Project Objectives

The goal of this project is to apply DevOps and Agile development practices by:

* Planning and managing work using GitHub Issues and Kanban boards
* Developing and testing a Flask-based REST API
* Implementing Continuous Integration (CI) using GitHub Actions
* Applying security best practices and automated testing
* Containerizing the application using Docker
* Deploying the application to Kubernetes
* Automating deployment using Tekton CI/CD pipelines

## Technology Stack

* Python
* Flask
* Docker
* Kubernetes
* GitHub Actions
* Tekton Pipelines
* OpenShift
